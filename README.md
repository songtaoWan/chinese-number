# convertToChineseNumber
将阿拉伯数字转换为中文数字，最大支持16位数字。
## install
```bash
$ npm install chinese-number --save
```

## example
```javascript
import convertToChineseNumber from 'chinese-number'

const num = '101'
convertToChineseNumber(num) // 一百零一
```
